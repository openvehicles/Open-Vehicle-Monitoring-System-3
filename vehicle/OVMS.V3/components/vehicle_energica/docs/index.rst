===================
Energica motorbikes
===================

Vehicle Type: **NR**

This vehicle aims to support all Energica motorbikes:
* EsseEsse9 (SS9)
* Eva Ribelle
* Ego
* Experia

It was tested on a Ribelle and Experia.
