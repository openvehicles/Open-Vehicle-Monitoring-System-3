## Notes

Remote clima session, 3 on/off cycles. First pass at the "hvac metric flips off while
running" bug.

> **SUPERSEDED by all-168388a82-…-224827.** This capture (car asleep, remote-only) made
> `0x5EA remote_mode` *look* like a clean run-state because the ECU was energized exactly
> during remote sessions. The follow-up ignition/ACC capture showed remote_mode is really
> "ECU energized" (pins with ignition off, asserted by radio mode) — NOT conditioning.
> Final source of truth is **`0x3B5 ClimaRunning`** (+ hold). See 224827 notes & `docs/vwegolf.dbc`.

Original (now-corrected) observations from this capture, still useful:
- `0x17332510` port-0x18 (`49 58 XX`) is a transactional ACK, not run-state — spurious
  `49 58 00` mid-session (t≈7.8, 219, 325). Never drives hvac.
- The spurious port-0x18 `00` was one half of the original flip-off bug.
